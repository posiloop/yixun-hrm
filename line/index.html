下方選單隨時可用，先試試【打卡】吧！
  1. 加入好友即觸發follow event → 推送歡迎訊息（greeting message）。
  2. 身分綁定引導輸入員編 / 掃店內 QR Code，將 LINE userId 對應到 HRM 員工資料。
  3. 綁定確認回讀姓名／分店確認無誤；未建檔者導向「聯繫人資」。
  4. 開通選單綁定後顯示六宮格圖文選單，推播「第一次打卡教學」。
  5. 日常推播班表異動、簽核結果、薪資條發放 → 由 HRM 通知中心推 LINE。
④ 開通清單

需人工開通的項目 待處理

以下需真人到 LINE 後台操作 / 提供，工程端拿到即可串接（我不接觸真實金鑰）。

🔑 需要提供 / 開通
  • LINE 官方帳號（LINE Official Account Manager 建立）
  • Messaging API channel（LINE Developers Console）
  • Channel access token ＋ Channel secret
  • LIFF app 建立並取得 {LIFF_ID}（Endpoint 指向 HRM）
  • Webhook URL（自動回覆後端，需一個可公開的 https 端點）
🛠️ 工程端可做（我來）
  • 圖文選單版型與熱區設定檔（依上表）
  • 關鍵字自動回覆邏輯（webhook handler）
  • 歡迎流程與員編綁定串接 HRM
  • 推播接到 HRM 通知中心（簽核/薪資/班表）
  • 多租戶：一店一 {LIFF_ID} / 一選單
